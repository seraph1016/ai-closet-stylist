# Requirements Document

## Introduction

AI 옷장 스타일 추천 웹사이트는 사용자가 보유한 옷을 등록하고, 상황별 코디를 추천받으며, 부족한 아이템에 대한 쇼핑 키워드를 제공하는 MVP 웹 애플리케이션이다. 사용자는 옷 정보를 직접 입력하며, 규칙 기반 추천 엔진이 코디 조합을 생성한다.

## Glossary

- **시스템**: AI 옷장 스타일 추천 웹 애플리케이션 전체
- **옷장_관리자**: 옷 등록, 수정, 삭제, 조회를 담당하는 모듈
- **추천_엔진**: 보유 옷 데이터를 기반으로 코디 조합을 생성하고 점수를 매기는 모듈
- **쇼핑_분석기**: 옷장 데이터를 분석하여 부족한 아이템을 찾고 검색 키워드를 생성하는 모듈
- **피드백_관리자**: 추천 결과에 대한 사용자 피드백을 저장하고 반영하는 모듈
- **의류_아이템**: 사용자가 등록한 개별 옷 정보 (카테고리, 색상, 계절, 스타일 태그, 메모, 착용 빈도 포함)
- **코디_조합**: 추천 엔진이 생성한 의류 아이템들의 조합 (top+bottom+shoes, top+bottom+outer+shoes, dress+outer+shoes)
- **카테고리**: 의류 분류 (top, bottom, outer, shoes, accessory)
- **상황_입력**: 코디 추천을 위해 사용자가 제공하는 정보 (occasion, weather, preferredStyle)
- **착용_빈도**: 사용자가 특정 의류 아이템을 착용한 횟수
